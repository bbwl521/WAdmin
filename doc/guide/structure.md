# 目录结构

```
WAdmin/
├── app/                    # 应用核心代码
│   ├── Controller/         # 控制器层
│   ├── Service/            # 服务层
│   ├── Repository/         # 数据仓库层
│   ├── Model/              # 数据模型层
│   ├── Middleware/         # 中间件
│   ├── Request/            # 请求验证
│   └── System/             # 系统核心模块
├── config/                 # 配置文件
├── databases/              # 数据库迁移与填充
├── plugins/                # 插件目录
├── public/                 # 入口文件
├── runtime/                # 运行时文件
├── storage/                # 存储目录
├── vendor/                 # Composer 依赖
├── web/                    # 前端代码
└── doc/                    # 文档站点
```
