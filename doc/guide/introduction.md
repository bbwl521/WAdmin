# 项目介绍

WAdmin 是基于 Hyperf 3.1 协程框架 开发的后台管理系统，采用 [Hyperf 3.1](https://hyperf.io) 协程框架构建。

## 技术选型

- **后端框架**: Hyperf 3.1 (PHP 8.1+)
- **依赖管理**: Composer
- **数据库**: MySQL 8.0+
- **缓存**: Redis
- **前端**: Svelte / Vue 3

## 核心特性

- 🔌 插件化架构，支持热插拔
- 🚀 协程高性能 HTTP 服务器
- 🏗️ 清晰的分层架构 (Controller → Service → Repository → Model)
- 🔐 内置 RBAC 权限管理
- 📝 代码生成器与 CLI 工具链
- 🗄️ 多数据库支持（插件独立数据库）
